This extension is built to be a test for a project I am involved in, where it has a feature to extract Google Meet caption text to be stored as AI input.

I built this extension as a way for myself to learn how to code in Vite + React, learn how to make a Google extension, and how to monitor HTML elements.

Most of the code is based off of this extension:
https://github.com/recallai/chrome-recording-transcription-extension
where they save the entire transcript as a text file to be saved.

The significant change between my extension and theirs is that mine aims to store the transcript into a textbox so the user can see the captions be written live.
